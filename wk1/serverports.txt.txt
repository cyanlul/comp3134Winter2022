What is the definition of server port?

A server port number is a way to identify a specific process to which an internet or other network message is to 
be forwarded when it arrives at a server.

What is server port range?

Port numbers range from 0 to 65535, but only port numbers 0 to 1023 are reserved for privileged services and 
designated as well-known ports.

How many bits does number above represent?

All 16 bits set will give you 65535.
